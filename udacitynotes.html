Important Concepts

The World Wide Web, How it Works, and HTML:

The Internet was created in the early 1990s, and has some 30 billion webpages at this time; which, if I'm being completely honest, surprised me quite a bit. Components of the web include: the user, you and your computer running a web browser such as Chrome (which is what I prefer), Safari, Firefox, and Internet Explorer, the Internet, and servers (computers like yours, but instead of browsing files, they are optimized for hosting files).

HTML stands for HyperText Markup Language and it is the basis, or foundation, for almost every webpage; it can include documents, images, videos, etc. HTML is the glue of connectivity that creates the "web" in which webpages are connected, or linked, by hyperlinks, making the web, "weblike." I was surprised to learn that there are some 30 billion pages on world wide web. Talk about the web that could be strung, with so many pages linking to each other.

Furthermore, HTML is made up of:
  • text content, or what you see
  • markup, or what it looks like
  • linkes to other pages, or hyperlinks
  • references to other documents, such as images, videos, etc.

    Tags, Elements, and Whitespace:

    Computers interpret language literally, so it makes sense that computer programming languages are designed to limit ambiguity. In HTML markup, tags typically bookmark the content of an element with an opening and closing tag. These tags help computers interpret your HTML with more specificity, giving your webpage more structure, and design. Tags can be used to bold, italicize, or underline words of text, for example.
